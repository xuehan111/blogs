RunTime是什么

OC是一门动态语言，其动态性体现在将数据类型的确定由编译时推迟到了运行时，这幕后的最大功臣就是Runtime。RunTime是一套比较底层的纯C语言API, 属于1个C语言库。我们平时编写的OC代码，在程序运行过程中，最终会转换成Runtime的C语言代码。
通过Runtime，我们可以了解到OC类的本质，方法的调用，以及消息转发机制。
